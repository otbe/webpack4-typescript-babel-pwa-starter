# Webpack4 + Typescript + Babel + PWA Starter

A simple (hopefully) state of the art starter.  
Lighthouse score 99 (perf)

## How to run

`$ yarn`  
`$ yarn watch`

Production build:  
`$ yarn build`

Use [serve](https://www.npmjs.com/package/serve) to serve your build:  
`$ serve -c 0 -s -p 8080 dist/`
